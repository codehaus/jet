General Use

Why JET - yet another open-source econometrics library?

While it is true that there are several open-source econometrics libraries available, most notably GRETL, none of these were written in Java, thus making the integration in Java applications very cumbersome (and nullying the benefit of cross-platform portability).

Another important differentiator of JET is that is released under a ASF license, and not under GPL, thus making it much more commercially-friendly.

System requirements

Which JDK is required to use JET?

1.4 or later.
