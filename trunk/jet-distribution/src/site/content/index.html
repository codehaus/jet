JET - Java Econometrics Toolkit

Aims and features

JET aims to provide a high-quality open-source (commercially-friendly) Java-based collection of econometrics tools.

Features of the first release will include:

  • Estimators of Multiple Linear Regression - using both OLS and GLS
  • Estimators and minimisers of Information Criterion
  • Estimators of critical values and p-values for several hypothesis tests.
